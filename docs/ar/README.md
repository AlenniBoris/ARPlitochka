## Текущий Продуктовый AR-Контракт

Android является продуктовым референсом для полного AR-flow:

1. Найти горизонтальные поверхности.
2. Поставить точки контура.
3. Замкнуть контур.
4. Подтвердить зону.
5. Показать заливку зоны.
6. Включить режим плитки.
7. Сменить тип плитки.
8. Повернуть текстуру плитки.
9. Сбросить зону / пересканировать.

iOS использует тот же shared state contract там, где это возможно, но рендерит через ARKit/SceneKit и native C/Obj-C bridge helpers.
