# :shared:core

Общий модуль для базовой инфраструктуры приложения.

## Ответственности:
- **Domain Models**: Базовые модели данных, используемые во всем приложении (например, `CustomResultModelDomain`, `CommonException`).
- **App Config**: Управление конфигурацией приложения (`AppConfig`, `AppConfigManager`).
- **Persistent Storage**: Настройка Jetpack DataStore для хранения настроек на разных платформах.
- **Common UseCases**: Общие сценарии использования, связанные с конфигурацией (например, включение моков).

## Правила:
- Модуль должен быть максимально независимым и не содержать бизнес-логики конкретных фич.
- Весь код должен быть KMP-совместимым.
- Платформенно-зависимый код (DataStore) реализуется через `expect/actual`.
