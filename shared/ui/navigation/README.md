# Модуль :shared:ui:navigation

Модуль управления навигацией и глобальными элементами навигации.

## Состав
- `BottomBarValues.kt`: Определение маршрутов (`CATALOG_ROUTE`, `AR_ROUTE`), константы `BOTTOM_BAR_ROUTES` и моделей для нижней панели.
- `AppBottomBar.kt`: Реализация кастомной нижней панели навигации с поддержкой анимаций и системных отступов.

## Ключевые принципы
- Все новые маршруты должны регистрироваться здесь.
- Видимость Bottom Bar управляется через список `BOTTOM_BAR_ROUTES`.
