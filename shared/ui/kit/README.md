# Модуль :shared:ui:kit

Набор переиспользуемых UI-компонентов (UI Kit) для всего приложения. Модуль структурирован по назначению компонентов.

## Структура и компоненты

### screens/ (Общие экраны состояний)
- `AppProgressScreen` — экран ожидания с анимированной иконкой.
- `AppExceptionScreen` — универсальный экран отображения ошибок с кнопкой повтора.

### ar/ (Компоненты дополненной реальности)
- `CenterReticle` — прицел для выбора точек на поверхности.
- `StatusPanel` — панель статуса и инструкций AR-сессии.
- `ArTopBar` — верхняя панель управления в AR.
- `ArContourActionButtons` — кнопки управления контуром.
- `BlockingMessage` — полноэкранные сообщения (ошибки AR, запросы разрешений).

### utils/ (Вспомогательные инструменты)
- `PlatformDebug` — инструменты для отладки на разных платформах.

## Особенности
- Поддержка **Compose Multiplatform**.
- Использование ресурсов из `:shared:ui:core`.
- Адаптация под Android и iOS.
