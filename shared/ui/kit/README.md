# Shared UI Kit

Набор переиспользуемых UI-компонентов для AR.

## Компоненты
- `ArReticle` — прицел для выбора точек на полу.
- `StatusPanel` — панель для отображения текущего статуса и инструкций.
- `BlockingMessage` — полноэкранные сообщения об ошибках или запросах разрешений.
- `DebugPanel` — панель для отображения отладочной информации.
