# Модуль :shared:ui:core

Базовый модуль для UI-слоя приложения. Содержит общие модели и ресурсы, не привязанные к конкретным визуальным компонентам.

## Состав
- **model/**: Общие UI-модели (например, `ExceptionModelUi`).
- **composeResources/**: Общие строковые ресурсы (ошибки, форматирование цен, единицы измерения) и изображения для всего приложения.
- Базовые сущности для работы с UI.
- Темы оформления (Material3).
- Общие расширения для Compose.

## Особенности
- Использует **Compose Multiplatform Resources**.
- Генерирует публичный класс ресурсов (`publicResClass = true`), доступный в других модулях.

## Зависимости
- `:shared:core` (для доступа к доменным моделям исключений).
- Только стандартные библиотеки Compose Multiplatform.
- Не должен зависеть от Feature-модулей или `:shared:ui:kit`.
