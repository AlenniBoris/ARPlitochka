# Module `:features:catalog`

Feature-модуль для отображения каталога плитки и выбора материалов для AR-примерки.

## Назначение

Модуль отвечает за:
- отображение списка доступной плитки;
- фильтрацию по категориям;
- выбор конкретной модели плитки для перехода в AR.

На текущем этапе экран является заглушкой с белым фоном.

## Структура

```text
features/catalog/
├── build.gradle.kts
└── src/main/java/com/example/arplitka/features/catalog/
    └── presentation/
        └── screen/
            └── CatalogScreen.kt
```

## Навигация

Экран каталога является стартовым экраном приложения и доступен через Bottom Navigation Bar.

## Ближайшее развитие

- реализация списка плитки с карточками (превью, название, размеры);
- интеграция с данными о плитке (TileType);
- переход в AR с предустановленной плиткой.
